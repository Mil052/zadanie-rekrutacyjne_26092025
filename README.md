# Zadanie rekrutacyjne
Projekt zasetupowany w Vite.

## Uruchomienie aplikacji lokalnie
1. Pobierz repozytorium
2. Zainstaluj _dependencies_  
  _npm install_
3. Uruchom lokalny serwer deweloperski  
  _npm run dev_
